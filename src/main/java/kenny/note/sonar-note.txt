在project的pom文件里面可以设置不进行sonar检测的code：
若仅仅设置skip=true，则会忽略掉整个project，
若继续指定了exclusions属性则会忽略掉特定的文件夹或文件。

<properties>
    <sonar.skip>true</sonar.skip>
    <sonar.exclusions>com/ericsson/ecds/bcc/m3/simulator/*</sonar.exclusions>
</properties>

文件路径可以采用模糊匹配，规则如下：
            ? 匹配单个字符
            ** 匹配0个或多个文件夹
            * 匹配0个或多个字符

关于路径的设置：
Don't start with "/src/main/java/" Instead start with "com/companyname/projectname/"
I personally prefer "**com/companyname/projectname/../**"

##############
Serializable的类的域也需要是Serializable的。若含有泛型参数，泛型参数也需要是Serializable的。
正确的示例：
class AAA implements Serializable{}

class BBB<T extends Serializable> implements Serializable{
    private List<AAA> contents; //不能有赋值（例如contents=null或其他都不行）
}
